  1. 定义一个名为 button-counter 的新组件

    在这个例子中组件的名字是 <button-counter>。我们可以在一个通过 new Vue 创建的 Vue 根实例中，把这个组件作为自定义元素来使用：

    在被Vue管理的标签中使用这个定义的组件，最终前端会渲染成button，而不被Vue管理的标签下使用组件时，不会生效。

    因为组件是可复用的 Vue 实例，所以它们与 new Vue 接收相同的选项，例如 data、computed、watch、methods 以及生命周期钩子等。仅有的例外是像 el 这样根实例特有的选项。

    组件种要用的参数应该由函数返回，即一个组件的 data 选项必须是一个函数，因此每个实例可以维护一份被返回对象的独立的拷贝：

有两个重要的生命周期函数：

  • beforeRouteEnter：在进入路由前执

  • beforeRouteLeave：在离开路由前执行

to：路由将要跳转的路径信息
from：路径跳转前的路径信息
next：路由的控制参数
next() 跳入下一个页面
next(’/path’) 改变路由的跳转方向，使其跳到另一个路由
next(false) 返回原来的页面
next((vm)=>{}) 仅在 beforeRouteEnter 中可用，vm 是组件实例

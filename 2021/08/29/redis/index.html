Posted on 2021-08-29 Edited on 2021-09-02 Views:

REmote DIctionary Server(Redis) 是一个key-value 存储系统，是跨平台的非关系型数据库。

官方测试Redis读取速度是110000次/s,写入速度是81000次/s

五大基本类型

字符串STRING

默认set的key都是String类型

  • 追加：

    可通过apped key appendValue的格式对原始的value进行追加。如果key不存在，会起到set的作用

  • 长度获取

    strlen key

  • ++/–

    incr key/decr key

  • +N/-N

    incrby key delta/decby key delta

  • 切片

    `getrange key start end，返回key[start]~key[end]，包含首尾。如果end是-1，则表示最后一个字符

  • 使用新的字符串替换指定下标开始的字符串

    setrange key startindex toreplace

  • 设置过期时间

    上面了解到expire key secondsToLive语法可以控制key的失效时间，并可以使用ttl key查看剩余失效时间。除此之外，还可以使用setex key secondsToLive 设置失效时间。

  • 不存在key则设置： setnx，若key不存在，则set，并返回1，若key已经存在，则返回0

  • 批量set/get

    mset/mget

  • 批量msetnx

    有一个失败，则全部set失败

  • getset

    感觉访客统计之类的工具就可以用这个实现

    如果key存在，则返回旧的key，并设置新的key；

    如果key不在，则返回nil，并设置key

  • 数组长度llen

有序集合ZSET

Redis 有序集合类似于 Redis 集合，也是一组非重复的字符串集合。但是，排序集的每个成员都与一个分数相关联，该分数用于获取从最小到最高分数的有序排序集。虽然成员是独特的，但可以重复分数。

三种特殊类型

地理位置geospatial

  • 增加地理位置数据

    有效的经度介于 -180 度至 180 度之间。

    有效的纬度介于 -85.05112878 度至 85.05112878 度之间。

    GEOADD key longitude latitude member [longitude latitude member ...]

  • 两地距离，可以携带参数指定单位

    如果两个位置之间的其中一个不存在， 那么命令返回空值。

  • 获取指定经纬度范围内的元素

    以给定的经纬度为中心， 返回键包含的位置元素当中， 与中心的距离不超过给定最大距离的所有位置元素。

  • 获取某元素附近某范围内元素

基数统计hyperloglog

基数:不重复元素。

大数有误差

PFADD PFADD key element [element …] 添加指定元素到 HyperLogLog 中。
PFCOUNT PFCOUNT key [key …] 返回给定 HyperLogLog 的基数估算值。
PFMERGE PFMERGE destkey sourcekey [sourcekey …] 将多个 HyperLogLog 合并为一个 HyperLogLog

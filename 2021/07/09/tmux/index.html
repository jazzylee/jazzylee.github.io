tmux

Posted on 2021-07-09 Views:

tmux用起来还是很简单的。

  1. 启动一个名为xxx的tmux会话

  2. 窗口创建与切换

    进入tmux会话之后，还可以创建窗口(tmux window)，依次按：

    • Ctrl+B
    • C

    可以发现tmux会话底部出现了新的名字，切换方法是，依次按：

    • Ctrl+B
    • #n

    当前处于活跃的窗口名后有(*)标识

  3. 离开会话

    • Ctrl+B
    • D

  4. 查看tmux会话列表

  5. 进入某个tmux会话

  6. 关闭会话

    直接在会话中Ctrl+D

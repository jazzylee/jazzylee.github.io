Hello, Java :)

入门篇

Java三大版本

  • JavaSE:标准版，桌面程序…
  • JavaEE:企业级开发，提供企业级的解决方案。如web，服务器等
  • JavaME:嵌入式开发

JDK,JRE,JVM

  • Java development kit，包含jre
  • java rumtime environment，包含jvm
  • jvm:底层虚拟机

第一个java程序

  • 创建Hello.java文件
  • 定义Hello类(与文件同名)

  • javac Hello.java=>生成Hello.class

  • java Hello

意外收获

  • ctrl+shift+esc:打开任务管理器
  • md directory_name:创建文件
  • rd directory_name：删除文件夹
  • 三高
    • 高可用
    • 高性能
    • 高并发

变量&常量

  • 类变量：类内static关键字修饰的变量

Scanner

next()对比nextline()

next

  • 读取到有效字符后才结束
  • 有效字符之前的空格自动去除
  • 有效字符后的空白作为分隔符或结束符
  • 不能得到带有空格的字符串

nextline

  • 返回回车之前输入的字符
  • 可以获得空格

IO流

流:内存与存储设备之间的数据通道

分类

  • 按方向
    • 输入流:读入内存
    • 输出流:将内存中的内容写到存储设备中
  • 按单位
    • 字节流:以字节为单位
    • 字符流:以字符为单位，用于读写文本数据
  • 按功能
    • 节点流:真正用于传输数据的流
    • 过滤流:增强功能，如缓冲流
